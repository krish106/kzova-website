Kzova Labs

A focused studio building Arion VPN and Windows software.

Kzova Labs is an independent software company building Arion VPN for Android and practical Windows software. The mission is simple software that respects users, works reliably, and explains its privacy model clearly.

Current focus

Focused products, better attention.

Kzova Labs puts product, design, backend, and monetization work into Arion VPN while also keeping Windows software as a real company product area.

Product belief

VPN actions should stay clean.

Connecting, disconnecting, and recovering from a failed route should not be interrupted by forced ads or confusing UI.

Principles

How the company makes product decisions.

Every visible feature should make the app easier to trust, easier to connect, or easier to understand.

Respect the user

Privacy copy, ad behavior, and app permissions should be direct and understandable.

Prefer reliability

Server filtering, retry behavior, and cache design matter more than decorative complexity.

Keep the app free

Ad revenue supports development, but rewarded value should be clear and optional.

What Kzova Labs builds today

Android VPN software and Windows desktop tools.

The current roadmap includes Arion VPN improvements plus Windows software work for clean, practical desktop workflows.

  • Arion VPN dashboard and server list UI.
  • Windows desktop software and utility workflows.
  • Backend route filtering and failure penalty logic.
  • Rewarded ad Time Bank Boost and retention flows.
  • Legal, app-ads.txt, privacy policy, and consent readiness.
